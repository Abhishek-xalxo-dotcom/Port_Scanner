# Port_Scanner

## Project Overview
Port_Scanner is a web-based network scanning tool that uses Nmap to identify open ports and running services on authorized targets.

## Project Objective
To demonstrate network reconnaissance techniques and controlled port scanning through a web interface.

## Key Features
- Scan IP addresses or domains
- Detect open ports and services
- Controlled scan parameters
- Web-based results display

## Technologies Used
- Python
- Flask
- Nmap (python-nmap)
- HTML, CSS, JavaScript

## Security Concepts Covered
- Network reconnaissance
- Port scanning
- Service detection
- Ethical hacking principles

## Project Workflow
1. User submits a target host.
2. Backend performs a controlled Nmap scan.
3. Scan results are parsed and displayed on the web UI.

## Disclaimer
This tool must only be used on systems you own or have explicit permission to scan.
